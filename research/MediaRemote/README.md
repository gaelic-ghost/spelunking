# MediaRemote Raw Research

Store raw local evidence for `MediaRemote.framework` here.

Good fits:

- symbol dumps
- generated headers
- SDK path notes
- command transcripts
- strings output
- `otool` and `nm` output
- runtime experiment logs
- temporary observations that are not ready for polished docs

Do not store secrets, personal account data, unrelated media-library data, or large generated artifacts that should stay outside Git.
